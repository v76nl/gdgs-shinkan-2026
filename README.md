# 新入生歓迎サイネージ (2026年度)

GDGoC Chuo 新入生歓迎イベント用のデジタルサイネージ

## プロジェクト構成

- `index.html`: アプリケーションのメイン画面
- `style.css`: レイアウトとアニメーションのスタイル
- `main.js`: 制御ロジック
- `signage.json`: スライドの表示データ
- `assets/`: 画像などの静的ファイル

## 特徴

- **自動再生**: スライドが一定間隔で自動的に切り替わります。
- **キーボード操作**: 左右キーでスライドを手動で切り替えることができます。
- **フェード効果**: 滑らかなフェードイン・アウトのアニメーション。
- **レスポンシブ**: ブラウザのサイズに合わせて表示されます。

## 使い方

1. `index.html`をブラウザで開きます。（ローカルサーバーの使用を推奨）
2. スライドの内容を変更するには、`signage.json` を編集します。
