# 新入生歓迎サイネージ (2026年度)

GDGoC Chuo 新入生歓迎イベント用のデジタルサイネージ

## プロジェクト構成

- `index.html`: アプリケーションのメイン画面
- `style.css`: レイアウトとアニメーションのスタイル
- `main.js`: スライド制御・描画ロジック
- `signage.json`: スライドの表示データ（テキスト・画像・iframeの構成）
- `assets/`: 共通で使用する画像などの静的ファイル
- `sketches/`: p5.js などのクリエイティブコーディング用HTMLファイル

## 特徴

- **自動再生 & キーボード操作**: 一定間隔での自動スライド切り替えと、左右方向キーを使った手動の切り替えに対応しています。滑らかなフェード効果を持っています。
- **動的カラーリング**: スライドの総枚数に応じて、最適なHueを均等に自動割り当てします。後からスライドを追加しても、全体で色が被ることなく綺麗に分散したカラーテーマ（淡い背景色・強調色）が設定されます。
- **インタラクティブアート埋め込み機能**: `iframe` をサポートしており、p5.js 等で作成されたクリエイティブコーディングをスライドの要素や背景として透過状態で表示できます。
- **外部データ構造**: 表示するテキストやレイアウトは全て `signage.json` で管理されています。

## 使い方

1. `index.html`をブラウザで開きます。
   ※ 外部のJSONファイルやiframeを読み込むため、**ローカルサーバー経由での起動を強く推奨**します。

2. スライドの内容を追加・変更するには、`signage.json` を編集します。

### スライドの色の指定について

`signage.json` 内の `style` プロパティで直接カラーコードを指定することも可能ですが、自動計算された動的カラーを使うことで洗練されたデザインが維持されます。以下のCSS変数を利用してください。

- `var(--slide-bg)`: そのスライド固有の淡い背景色
- `var(--slide-primary)`: そのスライド固有のメインカラー（文字やボーダーなど）
- `var(--slide-secondary)`: そのスライド固有のアクセントカラー（メインから少し色相をずらした色）

**記述例:**
```json
"style": {
    "color": "var(--slide-primary)",
    "borderBottom": "4px solid var(--slide-primary)"
}
```

### p5.jsスケッチ（アニメーション）の追加方法

フルスクリーンのリッチなアニメーションやインタラクティブな描画をスライドに追加できます。

1. `sketches/` フォルダ内に新しいp5.jsスケッチのHTMLファイルを作成します。（例: `my_sketch.html`）
2. 背景をスライドの動的な背景色と馴染ませたい場合は、HTMLのCSS設定で `background-color: transparent;` を指定し、p5.js側の `draw()` 内で `background()` の代わりに `clear();` を実行して透過させます。
3. `signage.json` に以下の要素データを追加します。

```json
{
    "type": "iframe",
    "url": "sketches/my_sketch.html",
    "style": {
        "top": "0",
        "left": "0",
        "width": "100%",
        "height": "100%",
        "zIndex": "0" 
    }
}
```

※ `zIndex` を小さく（`0`に）設定することで、前面に配置したテキストや画像の背景としてアニメーションを流すことができます。
