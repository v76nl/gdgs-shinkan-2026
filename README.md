# 新入生歓迎サイネージ (2026年度)

このプロジェクトは、新入生歓迎イベント用のデジタルサイネージです。

## プロジェクト構成

- `index.html`: アプリケーションのメイン画面
- `style.css`: レイアウトとアニメーションのスタイル
- `main.js`: スライドのデータと制御ロジック
- `assets/`: 画像などの静的ファイル

## 特徴

- **自動再生**: スライドが一定間隔で自動的に切り替わります。
- **キーボード操作**: 左右キーでスライドを手動で切り替えることができます。
- **フェード効果**: 滑らかなフェードイン・アウトのアニメーション。
- **レスポンシブ**: ブラウザのサイズに合わせて表示されます。

## 使い方

1. `index.html`をブラウザで開きます。
2. スライドの内容を変更するには、`main.js`内の `signageData` オブジェクトを編集します。
